观察以上数据，简要分析下：Â¶

  • #### 理解列名： PassengerId => 乘客ID，不可影响生还，忽略
    Survived => 获救情况（1为获救，0为未获救）
    Pclass => 乘客等级/船舱等级(1/2/3等舱位),这个较可能影响获救，权贵容易特殊照顾
    Name => 乘客姓名，感觉不能影响获救，忽略
    Sex => 性别，女性易受保护，有可能
    Age => 年龄，有可能
    SibSp => 堂兄弟/妹个数，有可能
    Parch => 父母与小孩个数，有可能
    Ticket => 船票信息，票编号 感觉不可能
    Fare => 票价，有可能，高价可代表着权贵
    Cabin => 客舱，有可能，因为可能距离救生艇的距离近容易获救
    Embarked => 登船港口，有可能，比如：发达的港口代表着地位不一样
所以，大概知道需要提取哪些列的信息了，有：Â¶
Pclass、Sex、Age、SibSp、Parch、Fare、Cabin、Embarked

def correlation(x, y):
    '''
    Fill in this function to compute the correlation between the two
    input variables. Each input is either a NumPy array or a Pandas
    Series.
    
    correlation = average of (x in standard units) times (y in standard units)
    
    Remember to pass the argument "ddof=0" to the Pandas std() function!
    '''
    std_x = (x - x.mean()) / x.std(ddof=0)
    std_y = (y - y.mean()) / y.std(ddof=0)
    
#     print std_x.head(), '--', len(std_x)
#     print std_y.head(), '--', len(std_y)
#     print (std_x * std_y).head(), '--', len(std_x * std_y)
    
    return (std_x * std_y).mean()
